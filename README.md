# flaskkit

Flaskkit is a collection of useful code snippets for Flask development in Visual Studio Code.

## Features
- Quick insertion of common Flask code snippets.
- Improved productivity for Flask developers.

## Installation
1. Open Visual Studio Code.
2. Go to the Extensions view by clicking on the Extensions icon in the Activity Bar on the side of the window.
3. Search for "flaskkit".
4. Click the Install button to install the extension.
5. Reload Visual Studio Code to activate the extension.

## Usage
1. Open a Python file in Visual Studio Code.
2. Start typing a Flask-related keyword (e.g., "!fapp").
3. Select the desired snippet from the IntelliSense suggestions.
4. The snippet will be inserted into your code.
